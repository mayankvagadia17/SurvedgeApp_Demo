# Survedge

Survedge is an Android geolocation app that calculates the distance between two points on a map. It allows users to collect GPS points, organize them into lines and projects, visualize them on an interactive map, and perform stakeout operations.

**Stack:** Kotlin · Android · Room · OSMDroid · MVVM · ViewBinding
