# Device Fragment

## Overview
`DeviceFragment` is the UI component responsible for managing and displaying hardware device connectivity status.

## Functionality
- **Entry Tab:** Occupies the first tab position in the system's `MainActivity`.
- **System Insets:** Dynamically adjusts its top padding to accommodate the system status bar, ensuring content is not obscured in edge-to-edge mode.
- **Role:** Designed to serve as the hub for GNSS/external device connection and status monitoring.
